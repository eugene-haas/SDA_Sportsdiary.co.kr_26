종목 메인 디자인 기획

종목 메인 시안을 위해atomic design이론에 맞춰 기획된 디자인 입니다. 자세한 내용은 상단의 메뉴를 참고해 주시기 바랍니다.

제이쿼리 3.1.1 (jQuery)

자바스크립트 작성에 도움을 주는 라이브러리 이며 각종 플러그인과 결합하여 UI 구현에 도움을 줍니다. SD 프로젝트의 메인 기술 중 하나이며 부트스트랩을 사용하기 전에 필수로 호출되어야 합니다.

에이잭스 (Ajax)

SD App을 이루는 핵심 기술 중 하나로 비동기 통신을 구현합니다. 개발 파트에서 주로 다루지만 작동원리는 꼭 알아두어야 합니다. jQuery와 더불어 곳곳에서 사용되고 있습니다.



자세히 보기

부트스트랩

SD App은 부트스트랩 기반으로 제작되어 있으며 modal, collapse 등의 기능이 많이 적용되어 있습니다. 반응형 적용에 도움을 주는 등 기본적인 틀을 잡아 주는 UI FrameWork 입니다.

자세히 보기

폰트 어썸(Font awesome)

폰트 형태의 아이콘을 적용하도록 해주는 라이브러리 입니다. 작은 이미지를 폰트 형태로 로드하여 사용자 성능 개선에 도움을 줍니다. SD App 곳곳에 사용되며 사이즈, 색상등은 font 속성으로 조절 해야 합니다.

자세히 보기

퓨전 차트 (Fusion Chart)

차트 구현에 도움을 주는 API 입니다. SD App 통계를 비롯한 각종 차트에서 쓰이고 있습니다. 그래프, 차트 기획시 사이트를 참고해 주시기 바랍니다. 기타 형태는 개발팀과 꼭 문의 바랍니다.

캘린더 (FullCalendar)

SD App 에서 사용되는 캘린더로 대회일지, 훈련일지 등에서 쓰이고 있으며 jQuery 1. 버전에 의존합니다. jQuery 3.1.1 버전을 통해 구현된 소스들과 충돌 위험이 있으므로 형태 변경, 다른 페이지 확장시에는 주의가 필요 합니다.

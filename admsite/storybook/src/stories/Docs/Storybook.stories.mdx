## 1. 기본적인 필요지식
### 1-1. html, css, javascript
 * 기본적인 퍼블리싱 능력은 필수입니다.
### 1-2. vue.js
 * 해당 앱은 vue-cli를 통해 만들어진 vue 앱입니다. [vue-cli 공식문서](https://cli.vuejs.org/guide/) 참고
### 1-3. webpack, javascript module
 * webpack의 기본적인 내용과 module에 대한 지식이 필요합니다. [webpack 공식문서](https://webpack.js.org/concepts/) 참고
### 1-4. storybook
 * 본 앱은 storybook이라는 프레임워크를 사용하여 만들었습니다. [storybook 공식문서](https://storybook.js.org/docs/react/get-started/introduction) 참고
### 1-5. +markdown 문법(.mdx)
 * 스토리북의 문서는 기본적으로 markdown 문법을 사용합니다. 기본적인 내용을 알아두세요. [markdown 설명](https://gist.github.com/ihoneymon/652be052a0727ad59601) 참고

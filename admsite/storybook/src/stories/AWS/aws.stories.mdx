## 사용중인 서비스

### 1. EC2(AWS Cloud Compute Service)
아마존의 가상컴퓨터를 임대 받아 쓰는 것<br/>
가상컴퓨터 1대 : 인스턴스 라고 부른다.

#### 비용
 * 프리티어 일 경우 **1달에 모든 인스턴스 사용량 총합이 750시간 초과**하면 비용부과
 * 프리티어 전용 인스턴스  세팅이 있으므로 주의

### 2. EBS(Amazon Elastic Block Store)
EC2에서 생성한 인스턴스의 실질적인 스토리지<br/>
한개의 인스턴스와 한개의 볼륨이 연결되어 사용된다.

#### 비용
 * 프리티어 일 경우 **총합 30GB를 넘으면** 비용부과
 * 인스턴스를 중지해도 볼륨은 살아있기 때문에 30GB에 계속 포함된다.(용량을 줄일려면 종료 혹은 삭제)

### 3. CodePipeline
지속적 통합 및 전달(CI/CD) 서비스<br/>
github와 연결해서 push감지 후 정해진 인스턴스에 배포

#### 비용
 * 프리티어 일 경우 **1개까지 무료**
 * **CodeBuild는 1달에 100분까지 무료**이니 주의
 * CodeDeploy는 EC2 인스턴스에 배포할 경우 무료
